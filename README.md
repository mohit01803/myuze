# myuzePlay Search API

Semantic search API powered by Sentence Transformers and Pinecone.

## Endpoints

- `POST /v1/search/semantic` - Semantic search
- `POST /v1/buckets/generate` - Generate content buckets
- `GET /health` - Health check
- `GET /test` - Quick test

## Environment Variables

Set `PINECONE_API_KEY` in Space settings.
